# Fruit Voting App 

A Python program that allows the user to vote for fruits given for multiple users and then if the user wants (it will also show the votes for individual fruit,shows the result also) the catch is , it allows 1 vote per name.

# How it Works

- Choose what you want from the options.
- Votes are saved.
- You can view total votes anytime and can also see the result.
- If select the exit option you now end the process.

# Features

- Dictionary based vote counting.
- Easy to extend with more fruits.

# Key Tech Used

-Python 3
-Dictionary
-List
-Loops
